## Run Locally

Clone the project

```bash
  git clone https://github.com/YoungC0DE/Notepad
```

Go to the project directory

```bash
  cd Notepad
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
